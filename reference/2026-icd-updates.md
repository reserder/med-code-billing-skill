## High-Subjectivity ICD-10 Areas FY2026

### Combination Codes
- Hypertension + Heart Disease + CKD: I13.xx series
- Diabetes + Complications: E11.xx (always link unless excluded)
- Sepsis + Organ dysfunction: A41.xx + R65.20/R65.21

### Sequencing Rules Reminders
- Etiology/manifestation: always code underlying disease first
- Neoplasm + complications: sequence based on reason for encounter
- Obstetrics: always use O codes when available for pregnancy
- Poisoning vs adverse effect vs underdosing: follow Table of Drugs

### Placeholder X Usage
- Required when code has fewer than 6 characters but 7th character needed
- Common in injury codes (S/T codes) and poisoning codes
- Forgetting X is a common coder error — always verify
